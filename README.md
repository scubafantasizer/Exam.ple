# exam.ple — AI Study Coach

> **Master any subject with a personal AI mentor that lives on your machine.**

**exam.ple** is a local-first, privacy-focused study management platform powered by Gemini. It’s designed for students who want to turn their study process into an organized, AI-driven journey. From tracking subjects to analyzing exam performance, **exam.ple** is your secondary brain.

---

## 🚀 Key Features

| Feature | Description |
|---|---|
| **🧠 AI Agent** | A fully autonomous agent that can create notes, add topics, and log mistakes directly via natural language chat. |
| **📚 Smart Topics** | Track your curriculum with visual progress indicators and status tracking (Not Started → Completed). |
| **📝 Exam Insights** | Log practice tests question-by-eye. Let the AI analyze your weak points and patterns. |
| **❌ Mistake Tracker** | A dedicated vault for wrong and blank answers. Take notes, correct them, and ensure you never fail the same way twice. |
| **🗒️ Dynamic Notes** | Markdown-ready notes, tables, and schedules—generated by you or the AI agent. |
| **📺 Resources** | Curate YouTube playlists and videos, automatically organized by topic. |
| **🌍 Bilingual** | Full Turkish and English support. Toggle instantly in settings. |

## 🛠️ Requirements

- **Node.js 18+** (Node 20+ recommended)
- **Gemini API Key** — [Get it free here](https://aistudio.google.com/app/apikey)
- **OS**: Linux, macOS, or WSL (Windows Subsystem for Linux)

## ⚡ Quick Start

```bash
# 1. Unzip and enter
unzip exam.ple.zip
cd exam.ple

# 2. Make the script executable
chmod +x scripts/start.sh

# 3. Launch
./scripts/start.sh
```

> **Note:** The first launch will automatically install dependencies and build the application. Subsequent starts are instantaneous.

## 📂 Project Philosophy & Structure

**exam.ple** is built on the philosophy of **Technical Sovereignty**. Your data stays in a local SQLite database, and the AI logic is transparent and extensible.

```text
exam.ple/
├── scripts/      # Distro-agnostic launchers
├── server/       # High-performance Express + TS Backend
│   └── src/
│       ├── routes/      # AI & Data handlers
│       └── ai-skills.md # The AI's cognitive core
├── client/       # Modern React + Vite + Tailwind 4 Frontend
├── shared/       # Shared Zod schemas & Drizzle ORM
└── data/         # Local SQLite storage (exam-ple.db)
```

---

## 🚢 Tech Stack

- **Core**: Node.js, Express, TypeScript
- **Frontend**: React 18, Vite, **Tailwind CSS v4**, TanStack Query
- **AI Engine**: Google Gemini 2.0 Flash
- **Database**: SQLite via Drizzle ORM
- **Visuals**: Lucide Icons, Framer Motion, Recharts

---

## 🌐 Language Support

The application is fully localized. You can switch between **Turkish** and **English** via the sidebar or Settings. The AI Agent's system prompt defaults to Turkish to provide a native-feeling coaching experience.
